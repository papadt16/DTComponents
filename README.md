# DTComponents

DTComponents is a small store app for selling electronic components at school. This is a front-end only prototype with an admin panel and checkout-to-PDF feature.

## Quick start
1. Clone the repo
```
git clone <your-repo-url>
cd DTComponents
```
2. Install dependencies
```
npm install
```
3. Run the dev server
```
npm run dev
```
4. Build for production
```
npm run build
```

## Admin
- URL: `/admin`
- Password: `dtadmin123` (change in `src/components/AdminPanel.jsx` before deploying if you want a different password)
